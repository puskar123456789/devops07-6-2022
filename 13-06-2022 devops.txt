2.What are Jar, War and EAR files? 

ANS- 1.JAR file stands for Java Archive- It's a file format based on the popular ZIP file format and is used for aggregating many files into one.
     2. WAR file stands for Web Application Resource or Web Application Archive-allows testing and deploying web applications .  
     3.EAR stands for Enterprise Application Archive-it is a compressed file that contains the libraries, enterprise beans,
        and JAR files that the application requires for deployment.
