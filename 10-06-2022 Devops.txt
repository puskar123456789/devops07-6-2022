1. What is encryption? 

ANS-Encryption is the method by which information is converted into secret code that hides the information's true meaning.
The two main kinds of encryption are symmetric encryption and asymmetric encryption.

3. What is SSL, http, https? 

ANS- 1.(HTTPS)-Hypertext Transfer Protocol Secure is a protocol that secures communication and data transfer between a user's web browser and a website.
     2.(HTTP)- HyperText Transfer Protocol, standard application-level protocol used for exchanging files on the World Wide Web.
     3.(SSL)-Secure Sockets Layer is a protocol for securing communication on the Internet.
-----------------------------------------------------------------------------------------------------------------------------------------------------------
4. What are network packets?.

ANS-A network packet is a small amount of data sent over Transmission Control Protocol/Internet Protocol (TCP/IP) networks.
The packet size is around 1.5 kilobytes for Ethernet and 64 KB for IP payloads.

7. What is Horizontal Scaling?

ANS-Horizontal scaling means by adding more machines to your pool of resources (also described as “scaling out”).
-----------------------------------------------------------------------------------------------------------------------------------------------------------
 
8. What is vertical Scaling?

ANS-vertical scaling means by adding more power (e.g. CPU, RAM) to an existing machine (also described as “scaling up”).
